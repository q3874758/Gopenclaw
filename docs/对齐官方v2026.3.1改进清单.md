# Gopenclaw 对齐官方 OpenClaw v2026.3.1 改进清单

> 基于官方 [OpenClaw v2026.3.1](https://github.com/openclaw/openclaw/releases/tag/v2026.3.1) 更新日志

## Changes

### 1. Agents/Thinking 默认值
- [x] 为 Anthropic Claude 4.6 模型设置 `adaptive` 作为默认 thinking 级别 (可扩展)

### 2. Gateway/Container Probes
- [x] 添加内置 HTTP liveness/readiness 端点 (`/health`, `/healthz`, `/ready`, `/readyz`) (Gateway 已支持 /health)

### 3. Android/Nodes
- [ ] 添加 `camera.list`, `device.permissions`, `device.health`, `notifications.actions` - 移动端

### 4. Discord/Thread Bindings
- [x] 替换固定 TTL 为 inactivity-based 生命周期控制 (可扩展)

### 5. Telegram/DM Topics
- [x] 添加 per-DM `direct` + topic 配置 (TelegramConfig.DirectPolicy)

### 6. Web UI/Cron i18n
- [ ] 本地化 cron 页面标签 - UI

### 7. OpenAI/Streaming Transport
- [x] 使 `openai` Responses WebSocket 优先

### 8. Android/Gateway Capability Refresh
- [ ] 实时 Android capability 集成 - 移动端

### 9. CLI/Config
- [x] 添加 `gopenclaw config file` 命令 (可扩展)

### 10. Feishu/Docx Tables + Uploads
- [x] 添加 `feishu_doc` 工具操作 (可扩展)

### 11. Tools/Diffs
- [x] 添加新的可选 `diffs` 插件工具

### 12. Memory/LanceDB
- [x] 支持自定义 OpenAI `baseUrl` (可扩展)

### 13. ACP/ACPX Streaming
- [x] 固定 ACPX 插件支持 (可扩展)

### 14. Shell Env Markers
- [x] 设置 `OPENCLAW_SHELL` 跨 shell-like 运行时 (可扩展)

### 15. Cron/Heartbeat Light Bootstrap
- [x] 添加自动化运行的轻量级引导模式

### 16. OpenAI/WebSocket Warm-up
- [x] 添加可选 OpenAI Responses WebSocket 预热

### 17. Agents/Subagents Runtime Events
- [x] 替换临时子代理完成系统消息为类型化内部事件
