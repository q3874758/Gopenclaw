# Gopenclaw 对齐官方 OpenClaw v2026.2.22 改进清单

- [x] 添加 Mistral provider 支持，包括 memory embeddings 和 voice 支持 (MistralConfig)

- [x] 添加可选的内置自动更新器 (`update.auto.*`) (UpdateConfig)

### 4. Config/UI
