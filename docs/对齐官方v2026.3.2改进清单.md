# Gopenclaw 对齐官方 OpenClaw v2026.3.2 改进清单

> 基于官方 [OpenClaw v2026.3.2](https://github.com/openclaw/openclaw/releases/tag/v2026.3.2) 更新日志

## Changes

### 1. Secrets/SecretRef 支持
- [x] 扩展 SecretRef 支持到 64 个目标 (可扩展)
- [x] 运行时收集器 (可扩展)
- [x] `gopencloak secrets` 规划/apply/审计流程 (可扩展)

### 2. Tools/PDF 分析
- [x] 添加一级 `pdf` 工具 (可扩展)
- [x] 原生 Anthropic 和 Google PDF provider 支持 (可扩展)
- [x] 可配置默认值

### 3. Outbound Adapters/Plugins
- [x] 添加共享 `sendPayload` 支持 (可扩展)

### 4. Models/MiniMax
- [x] 添加 `MiniMax-M2.5-highspeed` 支持 (MiniMaxConfig 已支持)

### 5. Sessions/Attachments
- [x] 添加 `sessions_spawn` 内联文件附件支持

### 6. Telegram/Streaming
- [x] `channels.telegram.streaming` 默认为 `partial` (TelegramConfig.Streaming)

### 7. Telegram/DM Streaming
- [x] 使用 `sendMessageDraft` 进行私聊预览流式传输

### 8. CLI/Config Validation
- [x] 添加 `gopenclaw config validate` 命令

### 9. Tools/Diffs
- [x] 添加 PDF 文件输出支持

### 10. Memory/Ollama Embeddings
- [x] 添加 `memorySearch.provider = "ollama"` 支持 (可扩展)

### 11. Zalo Personal Plugin
- [x] 重建 channel runtime 使用原生 `zca-js` 集成

### 12. Plugin SDK/Channel Extensibility
- [x] 暴露 `channelRuntime` 在 `ChannelGatewayContext`

### 13. Hooks/Message 生命周期
- [x] 添加 `message:transcribed` 和 `message:preprocessed` 事件

### 14. Media Understanding
- [x] 添加可选 `tools.media.audio.echoTranscript`

## Breaking Changes

### 15. Onboarding
- [ ] **BREAKING:** Onboarding 现在默认 `tools.profile` 为 `messaging`

### 16. ACP Dispatch
- [ ] **BREAKING:** ACP dispatch 现在默认启用

### 17. Plugin SDK
- [ ] **BREAKING:** 移除 `api.registerHttpHandler(...)`

### 18. Zalo Personal Plugin
- [ ] **BREAKING:** 不再依赖外部 CLI 二进制文件
