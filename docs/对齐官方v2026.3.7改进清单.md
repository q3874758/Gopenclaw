# Gopenclaw 对齐官方 OpenClaw v2026.3.7 改进清单

> 基于官方 [OpenClaw v2026.3.7](https://github.com/openclaw/openclaw/releases/tag/v2026.3.7) 更新日志

## Changes

### 1. Context Engine 插件接口
- [ ] 添加 `ContextEngine` 插件槽位
- [ ] 完整的生命周期钩子 (bootstrap, ingest, assemble, compact, afterTurn, prepareSubagentSpawn, onSubagentEnded)
- [ ] `LegacyContextEngine` 包装器
- [ ] `sessions.get` gateway 方法

### 2. ACP 持久化通道绑定
- [ ] Discord 通道和 Telegram topic 持久化绑定存储
- [ ] 路由解析
- [ ] CLI/文档支持

### 3. Telegram/ACP Topic 绑定
- [ ] 接受 Mac Unicode dash 选项前缀
- [ ] 支持 Telegram topic 线程绑定
- [ ] 路由绑定 topic follow-ups 到 ACP sessions
- [ ] 添加可操作审批按钮

### 4. Telegram/Topic Agent 路由
- [x] 支持 forum groups 和 DM topics 的 per-topic `agentId` 覆盖 (已添加 `TelegramConfig.Topics`)
- [x] 支持 Telegram topic 线程绑定

### 5. Web UI/i18n
- [ ] 添加西班牙语 (es) 本地化支持 - UI

### 6. Onboarding/Web Search
- [x] 添加 provider 选择步骤
- [x] SecretRef ref-mode 支持

### 7. Tools/Web Search
- [x] 切换 Perplexity provider 到 Search API
- [x] 新增语言/区域/时间过滤器

### 8. Gateway
- [x] 添加 gateway.auth.token 的 SecretRef 支持（通过环境变量扩展）

### 9. Docker/Podman 扩展依赖
- [x] 添加 `OPENCLAW_EXTENSIONS` 环境变量支持

### 10. 插件/Before Prompt Build
- [ ] 添加 `prependSystemContext` 和 `appendSystemContext`

### 11. 插件/Hook 策略
- [ ] 添加 `plugins.entries..hooks.allowPromptInjection`

### 12. Hooks/Compaction 生命周期
- [ ] 发出 `session:compact:before` 和 `session:compact:after` 事件

### 13. Agents/Compaction 后配置
- [ ] 添加 `agents.defaults.compaction.postCompactionSections`

### 14. TTS/OpenAI 兼容端点
- [x] 添加 `messages.tts.openai.baseUrl` 配置支持 (MessagesConfig.TTS)

### 15. Slack/DM Typing 反馈
- [x] 添加 `channels.slack.typingReaction` (SlackConfig.TypingReaction)

### 16. Discord/AllowBots 提及门控
- [x] 添加 `allowBots: "mentions"` (DiscordConfig.AllowBots)

### 17. Agents/工具结果截断
- [ ] 使用 head+tail 截断保留尾部诊断

### 18. Cron/任务快照持久化
- [ ] 跳过备份期间规范化持久化

### 19. Google/Gemini 3.1 Flash-Lite
- [ ] 添加 `google/gemini-3.1-flash-lite-preview` 支持
