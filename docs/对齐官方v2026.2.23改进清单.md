# Gopenclaw 对齐官方 OpenClaw v2026.2.23 改进清单

> 基于官方 [OpenClaw v2026.2.23](https://github.com/openclaw/openclaw/releases/tag/v2026.2.23) 更新日志

## Changes

### 1. Providers/Kilo Gateway
- [x] 添加一级 `kilocode` provider 支持 (KiloCodeConfig)

### 2. Providers/Vercel AI Gateway
- [x] 接受 Claude 简写模型引用 (VercelAIConfig)

### 3. Docs/Prompt Caching
- [ ] 添加专门的 prompt-caching 参考文档 - 文档

### 4. Gateway/HTTP Security Headers
- [x] 添加可选的 `Strict-Transport-Security` 支持 (GatewayConfig.HSTS)

### 5. Sessions/Cron
- [x] 硬化 session 维护
- [x] 添加 `session.maintenance.maxDiskBytes` (MessagesConfig.Maintenance)

### 6. Tools/web_search
- [x] 添加 `provider: "kimi"` 支持 (KimiProvider 已支持)

### 7. Media Understanding/Video
- [x] 添加原生 Moonshot video provider (可扩展)

### 8. Agents/Config
- [x] 支持 per-agent `params` 覆盖 (可扩展)

### 9. Agents/Bootstrap
- [x] 缓存 bootstrap 文件快照
