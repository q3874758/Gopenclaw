# Gopenclaw 对齐官方 OpenClaw v2026.2.3 改进清单

> 基于官方 [OpenClaw v2026.2.3](https://github.com/openclaw/openclaw/releases/tag/v2026.2.3) 更新日志

## Changes

### 1. Telegram TypeScript 清理
- [ ] 移除最后一个 @ts-nocheck，使用 Grammy types 直接 (Go 版本无需关注)

### 2. Onboarding
- [x] 添加 Cloudflare AI Gateway provider setup (作为搜索备选) - 可扩展
- [x] 添加 Moonshot (.cn) auth choice (已添加 KimiProvider)

### 3. Messages
- [x] 添加 per-channel 和 per-account responsePrefix overrides
- MessagesConfig 已支持 responsePrefix

### 4. Cron (大量改进)
- [x] 添加 announce delivery mode for isolated jobs
- [x] 添加 delivery mode config (CronConfig.DeliveryMode)
- [x] 默认 isolated jobs 为 announce delivery
- [x] 接受 ISO 8601 schedule.at
- [x] 硬迁移 isolated jobs 到 announce/none delivery
- [x] 默认删除 one-shot jobs after success
- [x] 抑制 announce delivery 期间的 messaging tools
- [x] 避免 duplicate deliveries

### Heartbeat
- [x] Heartbeat: allow explicit accountId routing for multi-account channels

### TUI/Gateway
- [x] TUI/Gateway: handle non-streaming finals
- [x] TUI/Gateway: refresh history for non-local chat runs

### Shell Completion
- [x] Shell completion: auto-detect and migrate slow dynamic patterns

### Telegram
- [x] Telegram: honor session model overrides in inline model selection

### Web UI
- [x] Web UI: fix agent model selection saves
- [x] Web UI: resolve header logo path when gateway.controlUi.basePath is set

### Onboarding
- [x] Onboarding: infer auth choice from non-interactive API key flags

### Security
- [x] Security: keep untrusted channel metadata out of system prompts
- [x] Security: enforce sandboxed media paths for attachments
- [x] Security: require explicit credentials for gateway URL overrides
- [x] Security: gate whatsapp_login tool to owner senders

### Voice Call
- [x] Voice call: harden webhook verification
- [x] Voice call: add regression coverage for anonymous inbound caller IDs

### Cron

- [x] Cron: deliver announce runs directly
