# Gopenclaw 对齐官方 OpenClaw v2026.2.13 改进清单

> 基于官方 [OpenClaw v2026.2.13](https://github.com/openclaw/openclaw/releases/tag/v2026.2.13) 更新日志

## Changes

### 1. Discord Voice Messages
- [ ] 发送带有 waveform 预览的语音消息 - 移动端
- [x] 支持本地音频文件发送（包括 silent delivery）

### 2. Discord Presence
- [x] 添加可配置的 presence status/activity/type/url (DiscordPresenceConfig)

### 3. Slack/Plugins Thread-ownership
- [x] 添加 thread-ownership outbound gating (通过 RoutingConfig)
- [x] Slack outbound hook wiring

### 4. Agents Model Support
- [x] 添加 synthetic catalog support for `hf:zai-org/GLM-5` (HuggingFaceConfig)

### 5. Skills
- [ ] 移除 duplicate `local-places` skill
- [ ] 保留 `goplaces` 作为单一支持的 Google Places 路径

### 6. Agents Context Diagnostics
- [x] 添加 pre-prompt context diagnostics (AgentConfig.ContextDiagnostics)
- [x] 显示 messages, systemPromptChars, promptChars, provider/model, session file

### 7. Hugging Face Provider
- [x] 添加 first-class Hugging Face Inference provider 支持 (HuggingFaceConfig)
- [x] 添加 onboarding auth choice/API key flow
- [x] 添加 default-model selection
