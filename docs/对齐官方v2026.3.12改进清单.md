## 需实现配置

### 新增配置项

```go
// AgentConfig 新增
type AgentConfig struct {
    // ... existing fields ...
    FastMode          bool   `json:"fastMode,omitempty"`           // 快速模式
    ServiceTier       string `json:"serviceTier,omitempty"`       // Anthropic service_tier
}

// Provider 配置扩展
type ProviderConfig struct {
    FastMode *ProviderFastModeConfig `json:"fastMode,omitempty"`
}
type ProviderFastModeConfig struct {
    Enabled  bool   `json:"enabled,omitempty"`
    Default  bool   `json:"default,omitempty"`
}
```
